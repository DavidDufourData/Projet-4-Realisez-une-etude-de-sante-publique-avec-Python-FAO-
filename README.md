# Réalisez une étude de santé publique avec Python

## Contexte et objectifs

![Logo FAO](P4_logo_FAO.jpg)  
Analyse des données de la FAO afin d’étudier les zones les plus touchées par la sous-nutrition dans le monde.  
L’objectif est de nettoyer les données, explorer les tendances, produire des visualisations et formuler des conclusions basées sur les indicateurs alimentaires.

## Soft Skills
- Réaliser une exploration complète de A à Z  
- Structurer une analyse claire et logique  
- Interpréter des tendances statistiques  
- Documenter efficacement les résultats  

## Compétences
- Nettoyer et préparer un jeu de données complexe  
- Manipuler Pandas pour explorer et transformer les données  
- Réaliser des visualisations avancées  
- Identifier tendances, anomalies et relations  

## Outils utilisés
- Python  
- Jupyter Notebook  
- Pandas, NumPy, Matplotlib, Seaborn
